# RMG Hire Portal

A premium, interactive, client-side web application designed to streamline resource management and hiring workflows for **Intelegencia**.

## 🚀 Key Features

*   **Unified Authentication Gateway**: Integrated Sign In and Sign Up gate with role-based routing (Admin/HR vs. Requester).
*   **Dual-Theme Interface**: Seamlessly switch between white and dark modes with persistent user preferences.
*   **Philippines-Specific Sourcing Channels**: Enhanced sourcing tracking for Philippine recruitment channels (e.g., JobStreet, LinkedIn).
*   **Cost-Per-Hire Analytics**: Dedicated cost tracking dashboard calculating hiring efficiency metrics.
*   **Interactive Analytics Dashboard**: Responsive, visual charts built for modern presentation needs.
*   **PPT Presentation Export**: Download ready-to-present PowerPoint slides directly from the analytics dashboard.
*   **Local Database Management**: Backup and restore all records client-side using `localStorage`.

## 🛠️ Setup & Running

This project runs entirely in the browser using HTML5, CSS3, and JavaScript, leveraging premium CDNs.

1. Clone or download the repository.
2. Open `index.html` directly in any modern web browser, or serve it using a local development server (e.g., Live Server in VS Code, `npx serve`, or python's `http.server`).
